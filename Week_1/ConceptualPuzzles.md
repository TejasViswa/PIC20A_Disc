# Conceptual Java Puzzles

## Question 1
What will be the Output of the below code:
```java
public class A {
    public static void main(String[] args)
    {
        if (true)
            break;
    }
}
```
Choices:

a) Nothing
b) Error

## Question 2
What will be the Output of the below code:
```java
public class A {
    public static void main(String[] args)
    {
        System.out.println('j' + 'a' + 'v' + 'a');
    }
}
```
Choices:

a) java
b) Something else (Other than simple concatenation)
