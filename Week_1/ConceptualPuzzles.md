# Conceptual Java Puzzles

## Question 1
What is the output of the following snippet?
```java
public class A {
    public static void main(String[] args)
    {
        if (true)
            break;
    }
}
```
Choices:
1. Nothing
2. Error

## Question 2
What is the output of the following snippet?
```java
public class A {
    public static void main(String[] args)
    {
        System.out.println('j' + 'a' + 'v' + 'a');
    }
}
```
Choices:
1. java
2. Something else (Other than simple concatenation)
